# Restaurant Management System

This project is a command-line restaurant management system developed in Python, using SQLAlchemy ORM for database persistence. The system simulates a real restaurant environment, allowing user registration, menu visualization, and order management with well-defined business rules.

Features
- Command-line interface (CLI) restaurant management system
- Automatic creation of default restaurants and menus
- Customer registration with data validation and club membership support
- Employee registration with role and salary management
- Order creation with multiple products per order
- Payment handling via Pix or Card
- Automatic delivery fee calculation based on payment method
- 10% discount applied to club members
- Order history visualization per customer
- Persistent data storage using a relational database

Technologies Used
- Python: backend logic and system automation
- SQLAlchemy: database modeling and ORM
- SQLite: persistent data storage
- Object-Oriented Programming (OOP)

Objectives
- Practice backend development using Python and SQLAlchemy
- Apply Object-Oriented Programming concepts
- Implement CRUD operations with relational database modeling
- Simulate real-world business rules in a restaurant system
- Strengthen logic, validation, and data persistence skills

Status: Project completed as a functional prototype for educational purposes.
